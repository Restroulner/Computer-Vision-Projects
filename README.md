# Computer Vision Projects

This repository is a collection of various computer vision projects and algorithms. It covers fundamental image processing techniques, object detection, image segmentation, and other advanced topics using libraries like OpenCV and TensorFlow/PyTorch.

## Projects Included:

-   **Image Filtering**: Implementations of common image filters (e.g., Gaussian blur, Sobel edge detection).
-   **Object Detection**: Examples using pre-trained models and custom training for object detection.
-   **Image Segmentation**: Techniques for semantic and instance segmentation.
-   **Facial Recognition**: Basic facial recognition system using OpenCV.

## Getting Started

To explore the projects, clone the repository:

```bash
git clone https://github.com/Restroulner/Computer-Vision-Projects.git
cd Computer-Vision-Projects
```

Each project will reside in its own directory with a dedicated `README.md`, Python implementation, and necessary assets.
